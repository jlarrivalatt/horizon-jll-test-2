How to create a new section in shopify
1. Find the sections folder
2. Right click and add new file
3. Name it"section__section-title.liquid"
4. Every section needs a schema with name and preset
```

{% schema %}
{
  "name": "Hero 2",
  "settings": [
    
  ],
  "presets": [
    {
      "name": "Hero 2"
    }
  ]
}
{% endschema %}
```
5. Go to shopify developer theme editor
6. add section to template and save
